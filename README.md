### データベースと連携したTodoアプリ
Todoの登録 / 削除 / 編集が可能

### 使用技術
- `React`
- [json-server](https://github.com/typicode/json-server)
    - [axios](https://axios-http.com)でAPI操作
- [Vite](https://ja.vitejs.dev)
    - `React`のプロジェクト作成で使用

### インストール
```bash
npm i -D json-server
npm i -D axios
```

### Vite開発環境立ち上げ
```bash
npm run dev
```

### json-server立ち上げ
```bash
npm run js
```

### その他
